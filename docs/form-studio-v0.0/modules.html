ChoiceOnSelected: { disable?: string[]; enable?: string[] }

Type declaration

    A list of group id, question id or choice id. They will be disabled when this choice is selected and enabled when this choice is unselected.

    A list of group id, question id or choice id. They will be enabled when this choice is selected and disabled when this choice is unselected.
